Steps to install and run RabbitMQ
1) download Erlang and RabitMq

2) start first Erlang and then rabit Mq

3) start the rabbit mq service by searching in windows 

4) run the below command to run :
C:\Program Files\RabbitMQ Server\rabbitmq_server-3.13.6\sbin>rabbitmq-plugins.bat enable rabbitmq_management

5) run with browser : http://localhost:15672
